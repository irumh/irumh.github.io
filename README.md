# Irum Hussain | Personal Portfolio

This is the source code for my personal portfolio website, built with Vite and vanilla HTML/CSS.

## Inspiration
- Design: [saeideh.com](https://saeideh.com/)
- Structure: [paperswithcode.com](https://paperswithcode.com/)

## Development
```bash
npm install
npm run dev
```

## Deployment
The site is built as a static site. Run `npm run build` and deploy the `dist` folder to GitHub Pages.
Note: Currently configured for raw source deployment if GitHub Pages is set to use the main branch root (common for username.github.io repos).
